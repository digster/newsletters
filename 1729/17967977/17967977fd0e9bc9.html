Public blockchains like Ethereum have opened the door for anyone with an internet connection to create and access digital markets. The resulting decentralized finance ecosystem ("DeFi" for short) has emerged as an alternative to the legacy global financial system, and has the potential to be more efficient, transparent, and democratized.

But a key challenge for legacy blockchains is to effectively scale as the demand has grown for the apps built on top of them. As more users have joined the DeFi ecosystem, the fees and transaction wait times have increased.

That's why Solana was developed. It's a next generation blockchain built to scale DeFi to global adoption, without sacrificing censorship-resistance or security.

📋 Task: Earn $1000 in USDC-SPL

Create a great Solana tutorial for a core primitive

An early DeFi ecosystem of apps has emerged on Solana to leverage its high speed and low fees. However, key documentation, tutorials, and other technical content needs to be created in order to improve the developer experience.

The Solana Labs team has built several core reference implementations of DeFi primitives:

  1. One for token swap/automated market maker functionality.
  2. One for a borrow/lend protocol.
