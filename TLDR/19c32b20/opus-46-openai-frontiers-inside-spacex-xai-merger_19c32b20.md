Anthropic releases Opus 4.6 with new ‘agent teams' (2 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Ftechcrunch.com%2F2026%2F02%2F05%2Fanthropic-releases-opus-4-6-with-new-agent-teams%2F%3Futm_source=tldrnewsletter/1/0100019c32b2057b-fb16b58a-173d-448f-b294-c988ef4e4b8c-000000/moDGr_Z4nig3IMD9KHI48yUFIrUmOzKtFR599pQgecQ=443)
Anthropic's latest version of Opus features Agent Teams, teams of agents that can split larger tasks into segmented jobs. It also comes with a one-million-token context window, allowing for the processing of larger documents. The segmenting of agent responsibilities allows them to coordinate work in parallel to complete tasks faster. Agent Teams is currently available in a research preview for API users and subscribers.

OpenAI Unveils Frontier, a Product for Building ‘AI Co-Workers' (3 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Flinks.tldrnewsletter.com%2FlIqwwr/1/0100019c32b2057b-fb16b58a-173d-448f-b294-c988ef4e4b8c-000000/ft8dnTuBZ4M0y5JyX22G9NIEgCPH6x4JfU8kHsbzC3w=443)
OpenAI Frontier is a new AI platform for building, deploying, and overseeing AI agents. It makes it easier for businesses to combine sources of data that agents need to perform tasks. The agents act as co-workers that can collaborate with humans and can be used alongside agents developed by other labs. Frontier is currently available to a limited set of customers. OpenAI has yet to disclose prices for the feature.

China is running the EV playbook on humanoid robots — and it's working (5 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Frestofworld.org%2F2026%2Fchina-humanoid-robots-unitree-agibot-tesla-optimus%2F%3Futm_source=tldrnewsletter/1/0100019c32b2057b-fb16b58a-173d-448f-b294-c988ef4e4b8c-000000/2zF6UEmEPwLXfOEJMOQT2oUw_E1SM-Ip38WiIwBIb8A=443)

Google hints at big AirDrop expansion for Android “very soon” (2 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Farstechnica.com%2Fgadgets%2F2026%2F02%2Fgoogle-hints-at-big-airdrop-expansion-for-android-very-soon%2F%3Futm_source=tldrnewsletter/1/0100019c32b2057b-fb16b58a-173d-448f-b294-c988ef4e4b8c-000000/GrJJ-omSCVyCLcwsgQnvztmTSJsTKNd9dptpA5nmTvQ=443)

Death of Software. Nah (13 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Flinks.tldrnewsletter.com%2F7KNVK5/1/0100019c32b2057b-fb16b58a-173d-448f-b294-c988ef4e4b8c-000000/lfqdNV70_np6StFv-Y8jbjdVU9KKseD5UQ-PANHSWLM=443)
