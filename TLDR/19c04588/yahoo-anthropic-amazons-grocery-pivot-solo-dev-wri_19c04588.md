Amazon to Shut Down All Amazon Go and Amazon Fresh Stores (2 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Flinks.tldrnewsletter.com%2FkjD1bG/1/0100019c04588677-fe47bfb3-9dc3-4bb4-837a-c6c0ed4c7a73-000000/3saJQDGbE2HtdRljCz0cXKi1pXy3lC4jpLVrOGz9xHI=442)

The Go and Fresh stores failed to deliver a distinctive customer experience and couldn't be scaled up. Some of the shuttered stores will be converted into Whole Foods stores. Amazon plans to open 100 new Whole Foods stores in the coming years.

Try text scaling support in Chrome Canary (7 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Fwww.joshtumath.uk%2Fposts%2F2026-01-27-try-text-scaling-support-in-chrome-canary%2F%3Futm_source=tldrnewsletter/1/0100019c04588677-fe47bfb3-9dc3-4bb4-837a-c6c0ed4c7a73-000000/Esse1JlS95EENdvMJCxe-jKZTxUhd_MdHdnflNGzV7s=442)
The new 'meta name=text-scale' HTML tag informs browsers that websites are designed to still work if users increase their OS text size. To support text scaling, developers should avoid overriding the initial font size, only use font-relative units for content, and test on devices to quickly figure out problem areas and think about how to adapt their layouts. The tag is currently supported in Chrome Canary with the 'Experimental Web Platform features' flag enabled. Support in other browsers will hopefully come later this year.

One Human + One Agent = One Browser From Scratch (2 minute read)
](https://tracking.tldrnewsletter.com/CL0/https:%2F%2Fsimonwillison.net%2F2026%2FJan%2F27%2Fone-human-one-agent-one-browser%2F%23atom-everything%3Futm_source=tldrnewsletter/1/0100019c04588677-fe47bfb3-9dc3-4bb4-837a-c6c0ed4c7a73-000000/D7JT6XXzeFgov2KkmGajSP2zh0hiblmPHbng2TqDphg=442)
one-agent-one-browser is a web browser built in three days with a single Codex CLI agent. It contains 20,000 lines of Rust with no Rust crate dependencies at all, with only OS system frameworks for image and text rendering. The browser successfully renders HTML and CSS and can even display SVGs and PNGs. The project proves that an agent driven by a talented engineer is enough to get a very solid basic renderer working.
