The Government of India's Ministry of Statistics and Program Implementation has created an impressive Model Context Protocol (MCP) to connect AI's to Indian datasets ( https://datainnovation.mospi.gov.in/mospi-mcp ) . An AI connected to data via an MCP essentially knows the entire codebook and can make use of the data like an expert. Once connected one can query the data in natural language and quickly create graphs and statistical analysis. I connected Claude to the MCP and created an elegant dashboard with data from India's Annual Survey of Industries. Check it out ( https://claude.ai/public/artifacts/870316d9-a8fe-49b1-a377-2bf0bf7ce809 ) .
